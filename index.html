P R O J E C T S

HurtAlert

Web application that finds the safest route from one place to another within Berkeley using police data. Also provides graphs and heat maps to illustrate crime trends in the city. Won the Code4Impact award at Angelhack Silicon Valley.

Emulations

My attempts at imitating popular mobile and web games. These include a console version of 2048, as well as iOS versions of Piano Tiles and Flappy Bird made using iOS Sprite Kit.

PokeLocate

Complementary app to the mobile game Pokemon Go. Solved an early navigational issue with the game by allowing users to post where they found Pokemon and share the information with others. iOS applicaiton made in Swift.

Illuminate

An original game I made where tiles gradually gain color and the player must tap tiles to turn them back to white. Game ends when all the tiles on the board are colored.
